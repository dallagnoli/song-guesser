# Song Guesser

CLI song/word guessing game written in C#.

## Features
- Dynamic song loading from `list.txt`
- Handles spaces and punctuation automatically
- Support for `Ctrl+Q` to quit and `Enter` for full-title guesses

## How to Run
1. Ensure you have the .NET SDK installed.
2. Clone the repo and navigate to the project folder.
3. Run the following command in your terminal:

```bash
dotnet run
